# Руководство пользователя — Textile Workshop ERP

## Вход в систему

1. Откройте https://textile-workshop-erp.com
2. Введите логин: `admin`
3. Введите пароль: `admin123`
4. Нажмите **«Войти»**

Для смены языка нажмите **RU / EN / KG** — доступны русский, английский и кыргызский.

---

## Навигация

После входа слева отображается боковое меню:

| Пункт | Что открывает |
|-------|--------------|
| Панель | Главный дашборд со статистикой |
| Заказы | Управление заказами |
| Клиенты | База клиентов |
| Склад | Учёт материалов |
| Производство | Ежедневные отчёты |

| Отчёты | Графики и аналитика |

В правом верхнем углу — переключатель языков, уведомления, имя пользователя и кнопка выхода.

## Дашборд (Панель управления)

Открывается автоматически после входа.

**Что показывает:**
- 4 карточки: выручка за месяц, всего заказов, активных заказов, низкий остаток на складе
- График выпуска продукции за неделю (сшито / упаковано)
- Таблица последних 4 заказов с клиентом, статусом и суммой

Чтобы перейти к заказам — нажмите **«Смотреть все»**.

## Работа с заказами

### Просмотр заказов
- Страница показывает таблицу всех заказов
- **Поиск** — введите имя клиента, название модели или номер заказа
- **Фильтр по статусу** — выберите в выпадающем списке (Все / NEW / SEWING и т.д.)

### Создание заказа
1. Нажмите кнопку **«Новый заказ»** (синяя, справа вверху)
2. В открывшемся окне:
   - **Клиент** — выберите из списка (если клиента нет — сначала добавьте на странице «Клиенты»)
   - **Модель** — выберите продукт (рядом видна цена)
   - **Количество** — сколько единиц нужно сшить
   - **Крайний срок** — дедлайн заказа
   - **Примечания** — особенности ткани, замеры, пожелания
3. Нажмите **«Сохранить заказ»**

Сумма заказа рассчитывается автоматически: `цена × количество`.

### Изменение статуса
В колонке **«Статус»** есть выпадающий список. Выберите новый статус:

```
Новый → В работе → Раскрой → Пошив → Упаковка → Отправлен → Завершён
```

Статус можно изменить в любой момент.

### Редактирование и удаление
- Карандаш (✏️) — редактировать заказ
- Корзина (🗑️) — удалить (с подтверждением)

## Работа с клиентами

### Просмотр
Клиенты отображаются карточками. На каждой карточке:
- Имя и аватар (генерируется по инициалам)
- Количество заказов
- Телефон и адрес
- Примечания

### Добавление клиента
1. Нажмите **«Новый клиент»**
2. Заполните:
   - **Имя / Название** (обязательно, минимум 2 символа)
   - **Телефон** (обязательно, формат: только цифры и +)
   - **Адрес** (необязательно)
   - **Примечания** (необязательно)
3. Нажмите **«Сохранить»**

### Поиск
Введите имя или номер телефона в строку поиска — список фильтруется в реальном времени.

## Работа со складом

### Просмотр материалов
Таблица показывает все материалы:
- Наименование (например, «Кашемир Премиум»)
- Единица измерения (метры, кг, штуки)
- Текущий остаток
- Минимальный порог — если остаток ниже, строка подсвечивается красным
- Цена за единицу
- Поставщик

Вверху — карточки со сводкой: всего позиций, критический остаток, стоимость склада.

### Добавление материала
1. Нажмите **«Новый материал»**
2. Заполните: наименование, единица, цена, количество, минимальный остаток, поставщик
3. Нажмите **«Сохранить»**

### Регистрация закупки
1. Нажмите **«Добавить закупку»**
2. Выберите материал из списка (рядом показан текущий остаток)
3. Введите количество закупленного и общую стоимость
4. Укажите дату
5. Нажмите **«Сохранить»**

Остаток материала на складе увеличится автоматически.

## Работа с производством

### Создание отчёта (левая панель)
1. **Заказ** — выберите из списка активных заказов (завершённые и отменённые не показываются)
2. **Сотрудник** — кто выполнял работу
3. **Дата** — дата смены (по умолчанию сегодня)
4. **Сшито** — сколько единиц сшил сотрудник
5. **Упаковано** — сколько упаковано
6. **Брак** — количество бракованных изделий
7. Нажмите **«Сохранить отчёт»**

### Что происходит при сохранении
- Материалы **автоматически списываются** со склада по техкарте продукта
- Если на складе недостаточно какого-то материала — появится ошибка с названием материала, отчёт не сохранится
- Данные попадают в расчёт зарплаты сотрудника

### Правая панель
- Общее количество отчётов и активных сотрудников
- Список всех отчётов (последние 10) с именем сотрудника, продуктом и количеством

## Работа с сотрудниками и зарплатами

### Список сотрудников (левая колонка)
Каждая карточка показывает:
- ФИО, должность
- Ставку за изделие (сом/шт)
- Статус: активен / неактивен

**Добавить сотрудника:**
1. Нажмите **«Добавить сотрудника»** или кнопку **«Новый сотрудник»**
2. Заполните: ФИО, телефон, должность, ставка за изделие
3. Нажмите **«Сохранить»**

**Редактировать:** нажмите иконку карандаша на карточке сотрудника.

### Расчёт зарплат (правая колонка)
1. Выберите **месяц** в поле даты справа вверху
2. Таблица автоматически покажет для каждого сотрудника:
   - Сколько изделий сшито за месяц
   - Сколько бракованных
   - Ставку за изделие
   - **Итого зарплата** = (сшито − брак) × ставка
3. Внизу — общая сумма к выплате и общий брак

### Пример расчёта
| Сотрудник | Сшито | Брак | Ставка | Зарплата |
|-----------|-------|------|--------|----------|
| Елена Ковальчук | 142 шт | 2 | 450 сом | (142-2) × 450 = **63 000 сом** |
| Давид Сафаллиев | 310 шт | 0 | 120 сом | 310 × 120 = **37 200 сом** |

## Отчёты

Страница с графиками:
- **Доходы и расходы** — столбчатый график по месяцам
- **Динамика производства** — линейный график (сшито + брак)
- **Статусы заказов** — круговая диаграмма (завершён / в работе / новый / отменён)
- **Топ продукция** — рейтинг продуктов по выручке с прогресс-баром

## Частые вопросы

### Как добавить новый продукт (модель)?
Через API: `POST /api/products` с телом `{"name": "...", "price": ..., "laborCost": ..., "description": "..."}`. В будущих версиях будет добавлена страница управления продуктами в интерфейсе.

### Как настроить техкарту (Bill of Materials)?
Техкарты настраиваются через базу данных (таблица `technical_card_items`). Каждая запись связывает продукт с материалом и указывает расход на единицу.

### Почему не создаётся отчёт производства?
Проверьте, что на складе достаточно материалов. Система показывает ошибку «Insufficient stock for material: [название]» если чего-то не хватает. Решение — зарегистрировать закупку нужного материала.

### Как сменить пароль?
В текущей версии смена пароля возможна через базу данных. В будущих версиях будет добавлена страница профиля.

### Система поддерживает несколько цехов?
В текущей версии — один цех. Мультицеховая поддержка запланирована как будущее улучшение.
